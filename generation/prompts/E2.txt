Write your own case study of a strategic situation using {{COMPANY}}. Analyse the case using theory/frameworks/analytical tools you have learnt on the module. The analysis should lead to a set of recommendations or action plan, to resolve the situation.

In producing this analysis, three properties matter: claims should be source-supported; the analysis should connect into a coherent chain; reasoning should be explicit.

Specifically:
- Each claim should carry an attributable source.
- Each claim should connect to the analysis rather than standing alone.
- Each inferential step should state why the evidence warrants the conclusion.
