# Coding Instrument — Withheld

The v5.3 coding instrument (claim extraction prompt, typing prompt, edge-extraction prompt, supporting codebook, and reconciliation framework) is **withheld from this public repository for IP reasons**.

The instrument is available to reviewers and editors under confidentiality. The corresponding paper states the public/private boundary explicitly.

## What this directory contains
- `HASHES.txt` — SHA-256 hashes of the frozen v5.3 coding prompt files. Proves immutability across the corpus build without revealing prompt content.

## What is published in this repository
- `../pre-registration/` — pre-committed analysis plan (Priority 0 timestamp).
- `../generation/` — generation prompts (TMBA/E1/E2/E3), methodology, and (pending) the generated AI reports.
- `../corpus/` — inclusion criterion, anonymised entity classification, composition counts, and audit log.

The corpus and the generation method are public; the coding instrument is private but reviewer-accessible.
