# Strategic Theatre — Public Transparency Repository

This repository accompanies a strategy paper on AI-generated case studies and reasoning under different elicitation conditions. It contains the pre-committed analysis plan, the generation methodology, the generated AI reports, the inclusion criterion + corpus composition, and an audit log of the corpus build.

**Public/private boundary**: the corpus and the generation method are public; the coding instrument is withheld for IP reasons but is available to reviewers and editors under confidentiality.

## Contents
- `pre-registration/` — the pre-committed analysis plan, dated and timestamped publicly before the confirmatory analysis.
- `generation/` (to be populated) — TMBA/E1/E2/E3 generation prompts, the methodology document, and the generated AI strategy reports.
- `corpus/` (to be populated) — inclusion criterion, anonymised entity classification, composition counts, and the build audit log.
- `coding/` (to be populated) — README explaining the instrument is withheld; HASHES of the coding prompts (to prove immutability) without revealing content.
